## Summary of Completed Work

This session implemented the modular architecture outlined in the January 8, 2026 briefing document. All critical items have been completed.

### 1. Multi-Author Bug Fix ✅
**Problem:** Books with multiple authors were indexed multiple times (once per author).

**Solution:** Changed SQL queries in `scripts/batch_index.py` to use `SELECT DISTINCT books.id` with Calibre ID as primary key instead of iterating over author-title pairs.

**Key changes:**
- `get_books_by_tag()`, `get_all_books()`, `get_books_by_author()` now return each book exactly once
- Added `_get_authors_for_book()` helper to aggregate authors as "Author1 & Author2 & Author3"

### 2. Hardware-Adaptive Profiles ✅
**Files created:**
- `src/archilles/hardware.py` - Hardware detection (CPU, RAM, GPU/VRAM)
- `src/archilles/profiles.py` - Three indexing profiles with metadata storage

**Profiles:**
| Profile | VRAM | Model | Device | Chunk Size |
|---------|------|-------|--------|------------|
| minimal | <6GB | bge-small-en-v1.5 | cpu | 1000 |
| balanced | 6-12GB | bge-base-en-v1.5 | cuda | 1000 |
| maximal | >12GB | bge-m3 | cuda | 1500 |

**Reference hardware:** Tom's ThinkPad P15 Gen 1 (64GB RAM, Quadro T1000 4GB VRAM) → "minimal" profile

### 3. Checkpoint System ✅
**Files created:**
- `src/archilles/indexer/checkpoint.py` - Book-level and chunk-level progress tracking
- Modified `scripts/progress_tracker.py` - Added `get_failed_books()`, `clear_failed_status()`

**Features:**
- `--retry-failed` - Retry only previously failed books
- `--list-failed` - Show failed books with error messages
- Session resume with clear options

### 4. Modular Parser/Chunker/Embedder Architecture ✅
**Parsers** (`src/archilles/parsers/`):
- `base.py` - `DocumentParser` ABC, `ParsedDocument`, `ParsedChunk`
- `registry.py` - Parser discovery and selection
- `pymupdf_parser.py` - PDF parser implementation

**Chunkers** (`src/archilles/chunkers/`):
- `base.py` - `TextChunker` ABC, `ChunkerConfig`, `TextChunk`
- `registry.py` - Chunker discovery
- `semantic.py` - Respects paragraph/sentence boundaries
- `fixed.py` - Fixed-size and token-based chunkers

**Embedders** (`src/archilles/embedders/`):
- `base.py` - `TextEmbedder` ABC, `EmbeddingResult`
- `registry.py` - Embedder discovery
- `bge.py` - BGE family (small/base/m3)

### 5. Integration Layer ✅
