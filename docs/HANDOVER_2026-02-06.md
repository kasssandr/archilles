## Neue Features

### 1. Footer-Seitenzahlen-Erkennung
**Datei:** `src/extractors/pdf_extractor.py`

- Erkennt Seitenzahlen sowohl im Header (erste 3 Zeilen) als auch Footer (letzte 3 Zeilen)
- Unterscheidet Seitenzahlen von Fußnoten-Markern:
  - Standalone Zahlen → Seitenzahl
  - Kleine Zahlen (1-20) mit Text → Fußnote
  - Mehrere Zahlen mit Kommas → Fußnoten-Referenzen
- Logging zeigt Quelle: `📄 Page labels: 285 extracted (250 header, 35 footer)`

### 2. Re-Indexierung fehlender Seitenzahlen
**Dateien:** `scripts/batch_index.py`, `scripts/find_books_missing_labels.py`

```bash
# Bücher identifizieren
python scripts/find_books_missing_labels.py

# Re-indexieren
python scripts/batch_index.py --tag "..." --reindex-missing-labels
```

### 3. Multi-Author Deduplizierung
**Datei:** `scripts/batch_index.py`

- SQL-Query verwendet jetzt `GROUP BY books.id` + `GROUP_CONCAT(authors.name, ' & ')`
- Bücher mit mehreren Autoren erscheinen nur einmal

### 4. --all Option
**Datei:** `scripts/batch_index.py`

```bash
python scripts/batch_index.py --all                    # Alle Bücher
python scripts/batch_index.py --all --skip-existing    # Nur neue
```

### 5. Stop-Word-Entfernung (12 Sprachen)
**Datei:** `scripts/rag_demo.py`

- EN, DE, FR, ES, IT, PT, NL, LA, RU, EL, HE, AR
- ~230 Stop-Words
- Automatisch bei Suche (außer bei `--exact`)
- Beispiel: "sociology of roman aristocracy" → "sociology roman aristocracy"

### 6. Confidence Threshold
**Dateien:** `scripts/rag_demo.py`, `scripts/web_ui.py`

- Neuer Parameter `min_similarity` (0.0-1.0)
- Filtert irrelevante semantische Ergebnisse
- Slider in Web-UI für semantic/hybrid Modus
- Interpretationen: Streng (≥0.7) / Ausgewogen (≥0.5) / Tolerant (<0.5)
